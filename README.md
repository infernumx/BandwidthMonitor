# BandwidthMonitor
Simple tool to track upload & download data in MB from program start


# Setup
> `pip install -r requirements.txt`

# Run
> `python main.py`

# Analysis
This will show graphs showing the data from the last dumped session
> `streamlit run analysis.py`
